# 2026-06-20 工作日志

## 修复三个关键 Bug

### 1. 初始化失败
- **根因**：`Memorial.init()` 从未被调用，`memorial.js` 的状态加载是异步的但没有 `await`
- **修复**：在 `app.js` 的 `DOMContentLoaded` 回调中添加 `Memorial.init()`；`memorial.js` 的 `init()` 方法加上 `async/await`

### 2. 祭奠堂按钮位置不对
- **根因**：`.memorial-actions` 在操作区顶部，不在人名之下
- **修复**：用 Node.js 脚本将按钮区从标题下方移到 `.ancestor-tablets` 之后

### 3. 族谱关系图无法显示
- **根因**：`tree.js` 的 `refresh()` 函数没有错误处理，`update()` 中的异常被静默吞掉，无任何报错
- **修复**：`refresh()` 添加 `.catch()` 错误捕获，控制台现在会输出错误信息
