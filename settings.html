⚙️ GitHub 仓库配置

需要 repo 权限。 点击创建 Token →

📋 使用说明

  1. 在 GitHub 创建仓库（建议设为 私有，族谱数据仅家人可见）
  2. 在仓库 创建 Personal Access Token，勾选 repo 权限
  3. 填写上方表单并保存
  4. 首次使用请点击「📥 初始化数据文件」在仓库中创建数据文件
  5. 配置会保存在浏览器本地，换电脑需重新填写

🔧 数据管理

🚀 部署到 GitHub Pages

  1. 确保仓库已推送最新代码（含 settings.html）
  2. 进入仓库 Settings → Pages → Source 选择 main 分支
  3. 访问 https://你的用户名.github.io/仓库名/
  4. 首次打开页面，点击右上角 ⚙️ 填写仓库配置
  5. 填写与你创建 Token 时相同的 GitHub 用户名
